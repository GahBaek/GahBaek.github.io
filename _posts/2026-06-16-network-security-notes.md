# Network Security Notes

This post is a template for organizing packet analysis notes.

## Packet Fields to Check

- Source and destination IP address
- Source and destination port
- Protocol
- Flags
- Payload size
- Timing and frequency

## Questions

- Is the traffic expected?
- Is the destination trusted?
- Is there repeated scanning behavior?
- Is sensitive data exposed?
