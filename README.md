# Personal GitHub Pages Template

Green and white personal academic/research portfolio template for GitHub Pages.

## How to use

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload all files in this folder to that repository.
3. Edit `_config.yml` and replace `Your Name` and `yourusername`.
4. Edit `index.md`, `about.md`, `cv.md`, `_data/projects.yml`, blog posts, and paper reviews.
5. Push to GitHub.
6. Open `https://<your-github-username>.github.io`.

## Local preview

Install Ruby and Bundler, then run:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Main content locations

- Personal information: `about.md`
- CV: `cv.md`
- Projects: `_data/projects.yml`
- Blog posts: `_posts/`
- Paper reviews: `_paper_reviews/`
- Design/CSS: `assets/css/style.css`
